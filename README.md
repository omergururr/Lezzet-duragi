# Lezzet Durağı - Restoran Web Sitesi

HTML, CSS ve PHP kullanılarak geliştirilmiş, modern bir arayüze ve dinamik içerik yönetimine sahip restoran web sitesi projesi.

## 🚀 Özellikler
* **Dinamik Menü Listeleme:** Restoran menüsündeki yemeklerin PHP altyapısı ile listelenmesi.
* **Modern Arayüz:** Kullanıcı dostu, şık ve responsive (mobil uyumlu) tasarım.
* **Yönetim Yapısı:** İçeriklerin dinamik olarak yönetilebilmesi için geliştirilmiş PHP sayfaları.

## 🛠️ Kullanılan Teknolojiler
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP

## 💻 Kurulum ve Çalıştırma

1. Bu projeyi bilgisayarınıza indirin (veya `git clone` yapın).
2. Proje klasörünü lokal sunucunuzun (`XAMPP/htdocs` veya `WampServer/www`) içine taşıyın.
3. Tarayıcınızdan `localhost/Lezzet-duragi` adresine giderek projeyi çalıştırın.
