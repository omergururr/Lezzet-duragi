# Lezzet Durağı - Restoran Web Sitesi

HTML, CSS ve PHP kullanılarak geliştirilmiş, dinamik içerik yönetimine ve modern bir arayüze sahip restoran web sitesi projesi.

## 🚀 Özellikler
* **Dinamik Menü Yönetimi:** Restoran menüsündeki yemeklerin listelenmesi ve yönetilmesi.
* **Modern Arayüz:** Kullanıcı dostu, şık ve responsive (mobil uyumlu) tasarım.
* **Yönetim Paneli:** İçeriklerin kolayca güncellenebilmesi için PHP tabanlı altyapı.

## 🛠️ Kullanılan Teknolojiler
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP
* **Veritabanı:** MySQL

## 💻 Kurulum ve Çalıştırma

1. Bu projeyi bilgisayarınıza indirin (veya `git clone` yapın).
2. Proje klasörünü lokal sunucunuzun (`XAMPP/htdocs` veya `WampServer/www`) içine taşıyın.
3. Tarayıcınızdan `localhost/Lezzet-duragi-website` adresine giderek projeyi çalıştırın.
